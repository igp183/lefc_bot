# Computational Physics Engineering Bot

A Discord bot for the Computational Physics Engineering course at UMa. 
Built and maintained by students.

## Features

- `/ping` — Check bot latency
- `/about` — Info about the bot
- `/resources` — Useful links for the course (filterable by category)
- `/roles` — Self-assign year and interest roles

## Want to Contribute?

## Quick Start (for running locally)

1. Clone the repo
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and paste your bot token
6. Run: `python main.py`

## Tech Stack

- Python 3.10+
- discord.py 2.x
- Hosted on Hetzner Cloud
